Ant has a default exclusion list for the most popular version control systems: CVS, Subversion and Visual SourceSafe. ClearCase is not part of this list.

So to fix this, add the following lines to your Ant script usually just after your tag. This only for Ant v1.6 and above.

<defaultexcludes add="**/.copyarea.dat" />
<defaultexcludes add="**/.copyarea.db" />

Not sure how to do the same with Maven yet though, but it should still be possible.
